Welcome to "An Asteroid As Big As"

A site inspired by at least three separate news articles with headlines that compare asteroids to random things.

If you would like to learn about this project or add your own silly thing of specific height, width, and length to the list of comparisons, check out this GitHub repository.

Volume is calculated using the formula: volume = 4/3 * pi * height * width * depth. For the asteroid we assume height is max asteroid diameter and minimum asteroid diameter is both width and depth. For the 'thing', we take the height, width, and depth and pretend the giraffe, gorrila, or whatever is also shapped like an ellipsoid. It is not perfect, but it is fair.
